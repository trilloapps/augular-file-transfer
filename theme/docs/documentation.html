Introduction

AdminPro Angular Dashboard Template is made with Angular 11+ CLI. This documentation will help you to understand template’s structure and how to edit. Please follow it very carefully to find out how template works and how you can edit things. You should have knowledge of Angular, CSS and HTML to be able to modify this template.

Before creating ticket, please make sure your question is regarding support only and not for custmaization and also follow full documentation to make sure the question / bug you are posting is not mentioned in documentation.

We at WrapPixel offers affordable custmaization services for our templates. For any custmaization work, please email us at info@wrappixel.com.

Please give 5 star rating to our template, as rating will help other people to trust our template.

Dependencies
  • Angular 11+
  • ng-bootstrap (with bootstrap 4+)
  • Sass
Browser Support

AdminPro angular is built to work in the latest desktop and mobile and tablet browsers.

  • Chrome (latest)
  • FireFox (latest)
  • Safari (latest)
  • Opera (latest)
  • Edge (latest)

Support
