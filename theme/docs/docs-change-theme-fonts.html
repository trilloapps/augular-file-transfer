How to Change Global Fonts ?

we have used google fonts in our template. so, when you change the fonts of the theme go tho the scss folder and open variable.scss file. In that we have import the google fonts link.
Note:
First run the demo you have and if you don't know how to run then follow this link project

ng serve -o

src assets scss variable.scss 

                                    
@import url('https://fonts.googleapis.com/css?family=Nunito+Sans:300,400,600,700,800');
                                    
                                  
                                    
$font-family-sans-serif: 'Nunito Sans',sans-serif !default;

src index.html 

                                                            						                                                                                    
<link
href="https://fonts.googleapis.com/css?family=Nunito+Sans:300,400,600,700,800&display=swap"
rel="stylesheet"
/>
Now, save your variable.scss and index.html[src] after that check in browser
If you want to change fonts of the theme then put the URL of your fonts and change font family in the variable.scss.
